### Architecture decisions made this session

- **Dashboards-as-code over UI-click-ops.** Specs in git, idempotent apply script, same philosophy as Dataform for transformations. README frames the YAML specs as source of truth; direct UI edits are drift.
- **URL-map path split for `/api/*` and `/embed/*`.** IAP gates `/*`; API-key and signed-JWT auth gate the API and embed paths respectively. Wildcard Host rule (`--new-hosts='*'`) prevents Host-header-spoofing routes around the split. Documented security tradeoffs in ARCHITECTURE.md — this is the kind of infrastructure-honest framing that belongs in a portfolio.
- **Looker Studio declared out of scope.** Metabase is the canonical BI tool. Simplifies the "one stack, one story" portfolio narrative.
- **Two-reader split on Q3 labeling.** REQUIREMENTS.md bullet reads "Campaign spend vs. revenue (ROAS)"; YAML `name:` and dashboard card label read "ROAS by campaign." Spec-doc reader sees what's shown; dashboard reader sees what's measured.
- **Friendly-key shape for `metabase-embed-config` secret.** 6b's Next.js signer reads `cardIds.funnel`, not `cardIds["Funnel conversion by channel"]`. Rename resilience, and the hard-coded mapping is unit-tested against the YAML `name:` fields.

### Gotchas for future sessions

- **Metabase API key is single-display-only.** Once generated in the UI, it cannot be retrieved again. If `metabase-api-key` in Secret Manager is lost, generate a new key and store as a new version; the old secret version is dead weight. README covers this.
- **`enable_embedding: true` on a card is a no-op until the global `enable-embedding-static` setting is flipped.** The README flags this; without it, 6b's signed embed URLs return 404.
- **`mart_channel_attribution` is monthly-grained.** A 30-day rolling window is not possible without rebuilding the mart at daily grain. The current "most recent month" language is honest about this constraint.
- **Dashcard reuse resolver has a latent duplicate-card limitation.** If a future dashboard places the same card twice (valid Metabase pattern), both positions get the same reused dashcard id and Metabase returns 400. Harmless today; comment flags it.
