### Architecture decisions made this session

- **Underside token set (`u-paper` / `u-ink` / `u-rule`) as a parallel scale** to the marketing-surface paper/ink tokens. Kept separate so future edits to either surface stay independent; consumed only inside the overlay.
- **Accent as a CSS variable (`--accent`) toggled by `OverlayProvider`** rather than React-prop-threaded. Any component can consume `accent-current` (Tailwind utility reading `var(--accent)`) without prop drilling.
- **Two-step boot phase state machine** (idle → boot → on): decouples the accent swap (130ms delay) from the panel-content reveal (260ms boot hold). CRT layers only render at `phase === 'on'`.
- **Tab-hinted open mechanism via `pendingTab`** rather than React Router or URL state. Keeps the overlay state machine self-contained while enabling deep-linking from the footer's three under-the-hood buttons.
- **Shared `useLiveEvents` hook** merging SSE (preferred, sticky once delivered) + dataLayer fallback. Single source of truth for the pipeline log feed, overlay timeline, and any future live-event visualization.
- **Honesty-pattern asymmetry across demo cards** is intentional: ", live" suffixes appear only where a paired "planned" bullet on the same card would otherwise create ambiguity. E-commerce has them (two live + one planned); subscription + lead-gen don't (declarative + roadmap-explicit).

### Gotchas for future sessions

- **Fonts sandbox:** `fonts.gstatic.com` is blocked in the Docker sandbox even though `fonts.googleapis.com` is reachable. `next/font/google` fails at build time. Bundle via `next/font/local` from the google/fonts GitHub repo if adding new fonts.
- **`.claude/agent-memory/` is subagent state** and should not be committed. Added to `.gitignore` this session after an accidental commit (immediately reverted via `96dbe71`).
- **jsdom `offsetTop` defaults to 0** on every element, which breaks the services scroll-spy's initial-sync path if run at `scrollY = 0`. Guard: only sync() when scrollY > 0 (documented in code comment).
- **`usePathname()` returns null in jsdom** without a router context. Header / services page / overlay all fallback via `?? '/'`.
- **Next.js App Router + `.ttf` fonts + `next/font/local`:** relative paths are resolved from the `fonts.ts` file location, not the project root. Paths go `../../public/fonts/...`.
- **`tsconfig target` pre-es2018:** the `/s` regex flag isn't compilable via `tsc --noEmit`. Use `[\s\S]*` instead of `.*` with the `/s` flag.
