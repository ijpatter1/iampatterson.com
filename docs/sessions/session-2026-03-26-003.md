2. **Consent spec/import documentation gap**, `web-container.json` spec says `"analytics_storage": "required"`, import file uses `"consentStatus": "NOT_NEEDED"`. Same behavior (Consent Mode v2 defaults handle enforcement), but the abstraction difference is not documented. (Important, documentation only)
3. **Contact form race condition**, `trackFormSubmit` pushes to data layer then immediately calls `router.push()`. SPA navigation doesn't trigger page unload so GTM should have time, but not guaranteed under heavy load. (Important, latent risk, not active bug)
4. **Design tokens defined but unused**, Semantic tokens in `tailwind.config.ts` not yet adopted by components (still using raw `neutral-*` classes). (Minor, migrate incrementally)
