## Session Notes

- **sGTM `ignoreUnknownValues: true` silently drops columns** — this was the root cause of missing demo params. Always verify raw table schema matches what the transport sends.
- **AI.CLASSIFY uses default connection** — no need to specify `CONNECTION` in the function call. Explicit `CONNECTION` syntax causes parse errors in some contexts.
- **URL-encoding from /g/collect** — all string values sent via GET params have `+` for spaces. Must decode in staging layer.
- **Cloud Run won't kill instances with in-flight requests** — `--no-traffic` stops new requests but old instances stay alive. Must delete service to force-kill.
